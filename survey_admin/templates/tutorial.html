Overview

This dashboard provides an interface for managing the kiosk database. Users can group questions together to form a survey, and deploy these surveys to a URL. The URL will be on display on a kiosk in a given building.

There are three main components of this dashboard:

  • Questions: Questions are linked to options. These questions are what will be shown on the kiosk.
  • Surveys: Surveys are a group of questions.
  • Deployments: Deployments link a URL to a survey. They also contain data regarding what building the given URL will be deployed in.

Add a Question

Please contact the database administrator to add a question.

Add a Survey

Add multiple questions to a survey by hitting the button and clicking on the questions you'd like on the survey. Once surveys are added, you cannot edit them.

Manage Deployments

You can add a deployment which links a URL to a survey. Only one survey may be added to a deployment at a given time. You may add a different survey to a given deployment by editing a deployment.

When should I add a deployment and when should I edit a deployment? For every device, there will be a unique URL. So, add a deployment when you have a new device. Edit a deployment when you already have a device with a URL but would like to show a different survey on it.
